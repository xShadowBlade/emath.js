Function calculateUpgrade

  • calculateUpgrade(value, upgrade, start?, end?, mode?, iterations?, el?): [amount: E, cost: E]

  • Calculates the cost and how many upgrades you can buy Uses inverseFunctionApprox to calculate the maximum affordable quantity. The priority is: target === 1 > costBulk > el

    Parameters

    • value: DecimalSource

      The current value of the currency.

    • upgrade: UpgradeStatic<string>

      The upgrade object to calculate.

    • Optional start: DecimalSource

      The starting level of the upgrade. Defaults the current level of the upgrade.

    • Optional end: DecimalSource

      The ending level or quantity to reach for the upgrade.

    • Optional mode: MeanMode

      The mode/mean method to use. See MeanMode

    • Optional iterations: number

      The amount of iterations to perform. Defaults to 15.

    • el: boolean = false

      ie Endless: Flag to exclude the sum calculation and only perform binary search. (DEPRECATED, use el in the upgrade object instead)

    Returns [amount: E, cost: E]

    [amount, cost] - Returns the amount of upgrades you can buy and the cost of the upgrades. If you can't afford any, it returns [E(0), E(0)].
